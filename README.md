# Anurag PC Sentinel VX

**Anurag PC Sentinel VX** is a real-time Windows PC health, security monitoring, and optimization desktop application.

It helps users monitor their PC performance, system health, CPU/RAM/Disk usage, battery condition, network activity, temperature information, and basic system events through a clean desktop dashboard.

## First-Time Setup

When the app opens for the first time, it asks for basic local setup details:

- Name
- Country
- State
- City
- Purpose
- Language
- PC Type
- Performance Mode

These details help the app customize the dashboard according to the user's PC type and usage.

The app does **not** ask for Email, Mobile Number, or OTP in the public version.

---

## Performance Modes

### Lite Mode

Best for low-end PCs.

Recommended for:

- 2 GB to 4 GB RAM PCs
- Older laptops
- Low-performance systems
- Users who want less CPU/RAM usage

### Balanced Mode

Recommended for most users.

This mode tries to balance smooth performance and useful monitoring.

### Performance Mode

Best for high-end PCs.

Recommended for:

- Gaming PCs
- Developer systems
- High RAM systems
- Users who want more complete monitoring

---

## PC Type Based Behavior

The app changes some dashboard behavior based on selected PC type.

### Laptop

Battery status will be shown.

### Desktop PC

Battery section will be hidden because most desktop PCs do not have a battery sensor.

### Student PC / Office PC / Gaming PC / Developer Workstation

The app adjusts monitoring behavior according to the selected performance mode.

## Hey Champ Voice Assistant

Anurag PC Sentinel VX includes an optional personal voice assistant called:

**Hey Champ**

If activated, users can say:

**Hey Champ**

Then speak commands such as:

- open chrome
- open notepad
- show health
- battery status
- system report
- open calculator

### Privacy Note for Voice Assistant

- Audio is not saved by PC Sentinel.
- Voice assistant is optional.
- If no microphone is detected, the assistant disables safely.
- Typed commands continue to work even if voice is disabled.

## Privacy and Data

Anurag PC Sentinel VX is designed to be local-first.

- User profile data is saved locally on the user's own PC.
- The app does not upload personal data by default.
- Email, phone number, and OTP are not required in the public version.
- Voice assistant is optional.
- No personal files are read automatically.
- Admin permission is requested only for advanced/admin actions.
